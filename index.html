Producto
Qué es el sistema

Un servidor MCP (Model Context Protocol) que se conecta a una instancia de Paperless-ngx para exponer toda la información de documentos almacenados — metadatos, contenido, etiquetas, corresponsales y tipos — como herramientas accesibles para cualquier cliente MCP compatible, incluyendo asistentes de IA como Claude.

02
Problema
Problema que resuelve
✕
Documentos almacenados en Paperless sin forma de consultarlos por lenguaje natural.
✕
Búsquedas manuales que requieren navegar la interfaz web o conocer la API REST directamente.
✕
Falta de integración entre el repositorio documental y herramientas de IA o automatización.
✕
Imposibilidad de cruzar información entre documentos de forma rápida e inteligente.

05
Herramientas
Herramientas MCP expuestas

El servidor expone un conjunto de herramientas que permiten al cliente MCP interactuar con Paperless de forma granular.

search_documents
Búsqueda full-text en documentos con filtros por etiquetas, corresponsal, tipo y rango de fechas.
get_document
Obtiene metadatos completos y contenido OCR de un documento específico por su ID.
list_tags
Lista todas las etiquetas disponibles en Paperless con su conteo de documentos asociados.
list_correspondents
Lista todos los corresponsales (remitentes/destinatarios) registrados en el sistema.
list_document_types
Lista los tipos de documento definidos para clasificación y organización.
download_document
Descarga el archivo original o la versión archivada (PDF) de un documento.

09
Riesgos
Riesgos y mitigación
Riesgos Identificados

Exposición no autorizada de documentos sensibles a través del protocolo MCP si no se configuran controles de acceso adecuados.

Mitigación
  • Autenticación por token con permisos granulares
  • Transporte cifrado TLS entre todos los componentes
  • Filtro de documentos por permisos de usuario
  • Logs de auditoría en cada consulta MCP
